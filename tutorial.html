1. 环境准备

确保你已经搭建好 GitHub Pages 或本地服务器，并具备基础的 HTML / JSON 编辑能力。

2. JSON 数据采集

影片信息统一写入 movies.json，包括标题、分类、日期、封面、大小、分辨率和片长等字段。

{
  "title": "示例影片 001",
  "category": "中文字幕",
  "date": "2025-08-18",
  "image": "images/movie1.jpg",
  "link": "#",
  "size_gb": 4.2,
  "resolution": "4K",
  "duration": 125
}

3. 页面渲染

前端通过 fetch("movies.json") 动态加载影片数据，并支持搜索、分类筛选和分页展示。

4. 更新与维护

只需更新 movies.json 文件即可完成影片更新，无需修改 HTML。

5. 常见问题

如果 GitHub Pages 部署后出现 fetch 读取不到 JSON，请确认 movies.json 路径与页面在同一目录。
